# Python Portfolio 🚀

This repository showcases Python-based data projects focused on practical analytics, financial modeling, and exploratory data science.

## 📁 Projects Included

### 1. 📊 **Portfolio Optimisation**
A simulation-based project exploring how to construct an equal-weighted portfolio and identify the optimal risk-adjusted allocation using the Sharpe Ratio.

**Features:**
- Historical equity data (AMD, AAPL, MSFT, ORCL)
- Daily returns and normalized performance tracking
- Monte Carlo simulation of 10,000 portfolio weight combinations
- Sharpe Ratio optimization and efficient frontier visualization

**Tech Stack:**  
`Python`, `Pandas`, `NumPy`, `Matplotlib`, `Jupyter Notebook`

📂 [`PortFolio Optimisation/`](./PortFolio%20Optimisation)

---

### 2. 🛒 **Retail Sales Analysis**
Exploratory analysis of retail sales transaction data to uncover trends, segment performance, and revenue drivers.

**Focus Areas:**
- Revenue breakdown by product, city, and store
- Outlier detection using IQR
- Visual storytelling through Matplotlib

📂 [`Retail Sales Analysis/`](./Retail%20Sales%20Analysis)
